# Dreamlandia-Project

### Team Members
Suchanut Namcharoen, Sami Hamdalla, Leonardo Rodriguez, Courage Idahosa, Yongfu Zeng

## Set Up
 Clone the repo
```
git clone https://github.com/suenam/Dreamlandia-Project.git
```

Install dependencies for the frontend and backend
```
cd client
npm i
```
```
cd server
npm i
```
## Running the project
- Running the server: 
```
cd server
npm run dev
```
- Running the client: 
```
cd client
npm run dev
```
